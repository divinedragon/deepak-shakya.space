{repository}[Boeing/config-file-validator] - Single cross-platform CLI tool to validate different configuration file types.

image::{hero-image}[Config File Validation demo]

== Supported formats:
* Apple PList XML
* CSV
* EDITORCONFIG
* ENV
* HCL
* HOCON
* INI
* JSON
* Properties
* TOML
* XML
* YAML

If you want to deploy this validation as part of Github Actions, use {github-action}[kehoecj/validate-configs-action]
