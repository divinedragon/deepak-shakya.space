{website}[CSS Grid Garden] - Grid Garden is a game for learning CSS grid layout.

image::{hero-image}[CSS Grid Garden Home Page screenshot which shows text code on the left and 9x9 grid garden on right]

Grid Garden is an interactive browser-based game designed to help developers and students master the fundamentals of the CSS Grid layout system. Players must write real CSS code to water carrot crops and poison weeds, effectively learning how to position elements and define grid structures through hands-on practice.

Across 28 increasingly challenging levels, the game provides a visual, gamified environment that makes complex web design concepts like grid-columns, grid-rows, and repeat functions much easier to grasp.
