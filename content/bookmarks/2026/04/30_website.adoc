{repository}[rga a.k.a ripgrep-all] is a line-oriented search tool that allows you to look for a regex in a multitude of file types. rga wraps the awesome {ripgrep}[ripgrep] and enables it to search in pdf, docx, sqlite, jpg, movie subtitles (mkv, mp4), etc.

Here is an example directory with different file types:

[source, bash]
----
demo/
├── greeting.mkv
├── hello.odt
├── hello.sqlite3
└── somearchive.zip
├── dir
│ ├── greeting.docx
│ └── inner.tar.gz
│ └── greeting.pdf
└── greeting.epub
----
