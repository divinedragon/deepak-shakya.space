{website}[svgl.app] - A beautiful library with SVG logos. Built with Sveltekit & Tailwind CSS.

image::{hero-image}[svgl app screenshot]

The platform allows developers to easily search for, download, or copy the code of optimized vector logos for their web projects. Beyond the web interface, it offers a public API and a variety of community-built extensions for Figma, VS Code, and React to streamline the integration of brand icons into any workflow.
